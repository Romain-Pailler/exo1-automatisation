# React + Vite

## Installation et Lancement

1. **Installer les dépendances** :
   ```bash
   npm install

2. **Lancer le serveur en local**:
    ```bash
    npm run dev #acces au localhost 3000

3. **Lancer le serveur de développement**:
    ```bash
    npm run build 

4. **Lancer la version construite**:
    ```bash
    npm run preview
